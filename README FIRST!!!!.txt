	a. raw
	b. hsv
	c. filtered
	d. result
	e. video
4. Pastikan video input berada dalam satu directory atau folder dengan main.py
5. pastikan telah menginstal package numpy dan cv2 (opencv-phyton)
6. Program ini bagus dijalankan pada phyton versi 3.7 ke atas
